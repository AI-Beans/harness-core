# Progress Log

> Each agent session appends an entry here before ending.
> This is the primary handoff mechanism between sessions.

## Format

```
### YYYY-MM-DD HH:MM — Session: <ID>

**What I did**: ...
**Key decisions**: ...
**What the next agent should do**: ...
**Verification status**: pass / fail (details)
```
