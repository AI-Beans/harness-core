## Getting Started

1. Initialize your project with `uv init` or similar
2. Copy `.harness/` and `docs/` to your new project root
3. Customize `progress.txt` and `feature_list.json` for your use case
4. Run `bash .harness/verify.sh` to start the verification loop

## Template Variables

When starting a new project, replace:
- `Self-Evolving AI Lab` → Your project name
- `agent@self-evolving.ai` → Your git identity
- Feature list tasks → Your project's specific tasks
